# OA办公管理系统 java457

## 项目概述

OA办公管理系统是基于ssm+vue技术栈的Javaee项目。该项目采用经典MVC三层架构，结合Spring、SpringMvc、Mybatis框架，以及Vue.js和Maven构建工具，实现了前端与后端分离的Web应用。系统使用MySQL数据库作为数据存储，界面友好，操作便捷。

## 技术栈

- 后端：Spring + SpringMvc + Mybatis
- 前端：Vue.js
- 构建工具：Maven
- 数据库：MySQL

## 功能模块

- 用户管理
- 文档管理
- 工作流程管理
- 考勤管理
- 报告生成

## 系统角色

- 系统管理员
- 普通用户
- 高级用户

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL（建议版本5.7以上）
- JDK版本：1.8以上
- Web服务器：Tomcat（建议版本8.5以上）

## 部署步骤

1. 安装并配置好MySQL数据库
2. 导入项目提供的数据库文件
3. 部署应用到Tomcat服务器
4. 启动Tomcat，访问前端页面进行操作

## 目录结构

```
OA办公管理系统

│   ├── src
│   ├── pom.xml
├── front-end（前端代码）
│   ├── src

```

## 核心亮点

- 前后端分离设计，便于维护与扩展
